# uff-pv-thermal-dataset
Dataset UFF-PV-Thermal: Imagens termográficas de módulos fotovoltaicos capturadas por VANT, em conformidade com a IEC 62446-3. Inclui imagens de defeitos como pontos quentes e falhas de diodo. Desenvolvido para o treinamento de modelos de visão computacional na inspeção automatizada de usinas solares.
